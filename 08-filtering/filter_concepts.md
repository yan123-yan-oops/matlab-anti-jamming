# 数字滤波概念解析

> 对应 C 的学习：C 里你写 for 循环做平均 → MATLAB 里用 filter 函数

---

## 1. 滤波的本质

```
滤波 = 让你想要的频率通过，拦下你不想要的频率
```

就像筛子：
```
低频成分（信号）  →  ✅ 通过（筛眼大）
高频成分（噪声）  →  ❌ 拦住（筛眼小）
```

## 2. FIR vs IIR

| | FIR | IIR |
|--|-----|-----|
| 全称 | 有限冲激响应 | 无限冲激响应 |
| 反馈 | 没有反馈 | 有反馈 |
| 相同性能下的阶数 | 高（计算量大） | 低（计算量小） |
| 相位 | 线性（不畸变） | 非线性 |
| 嵌入式实现 | 简单，就是乘累加 | 需要注意稳定性 |

**嵌入式里怎么选**：
- 要求线性相位（如音频）→ FIR
- 资源受限（如 MCU）→ IIR
- 简单滤波（如抗干扰 Notch）→ IIR，阶数低

## 3. 移动平均 = 最简单的 FIR

```
y[n] = (x[n] + x[n-1] + x[n-2] + x[n-3] + x[n-4]) / 5

这个就是 C 代码：
for (int i = 0; i < 5; i++) sum += buf[i];
out = sum / 5;
```

## 4. 陷波器 = 干掉特定频率

```
陷波器 = 带阻滤波器的特例（阻带非常窄）

北斗抗干扰：
  信号在 4.098MHz 附近
  干扰在 4.398MHz（偏移300kHz）
  → 设计陷波器在 4.398MHz
  → 信号几乎不受影响，干扰被干掉
```

## 5. C 代码实现滤波 vs MATLAB

```
MATLAB:    y = filter(b, a, x);             ← 一行搞定
C 代码:    for (i=0; i<N; i++) {            ← 要手写
               y[i] = b0*x[i] + b1*x[i-1] 
                    - a1*y[i-1] - a2*y[i-2];
           }
```

面试时能说出这个区别 → 说明你懂两边的差异
