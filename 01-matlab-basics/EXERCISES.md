### ✅ 概念题

1. MATLAB 的索引从几开始？和 C 有什么不同？
   MATLAB的索引是从1开始的，而C语言的索引是从0开始。因为MATLAB中的数据结构是以矩阵为基础数据结构的，所以一维向量第一个元素索引为1。
2. `v(end)` 是什么意思？`v(2:2:10)` 呢？
   v在MATLAB中代表的含义是向量，而v（end）呢就是v这个向量的最后一个元素。v(2:2:10)的意思是[2，4，6，8，10]这个向量。
3. 向量化运算比 for 循环快在哪？什么时候不能向量化？
   向量化运算在MATLAB中比C的经典for循环更快，因为向量运算每一次不需要重复的遍历过程，而是直接一次性算好。*
4. `uint8(300)` 输出什么？和 C 里的 `(uint8_t)300` 一样吗？
    无符号8位整型0-255，所以得300-256 = 44.和C中的输出一样。
5. `clear; close all; clc;` 分别做什么？
   clear是清空工作区所有的变量，可以加参数来指定删除变量。close all是关闭所有的figure，clc是clear command window，仅仅只清除命令行的所有信息。
