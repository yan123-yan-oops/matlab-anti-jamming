## 🔴 致命错误

### 1. 采样率必须满足奈奎斯特定理

```matlab
% ❌ 错误：t = 1:10 只有10个点，对10Hz信号完全不够
t = 1:10;
y = sin(2*pi*10*t);    % 这会产生严重混叠

% ✅ 正确：采样率至少 > 2×最高频率
Fs = 1000;
t = 0:1/Fs:1;
y = sin(2*pi*10*t);
```

### 2. 浮点数不能用 %d 打印

| 占位符 | 用途 | 示例 |
|--------|------|------|
| `%d` | 整数 | `printf("%d", 42)` |
| `%f` | 浮点数 | `printf("%.4f", 3.1415)` |
| `%s` | 字符串 | `printf("%s", "hello")` |

### 3. mean() 的返回值类型

```matlab
r = rand(5,5);
mean(r, 1)   % 返回 1×5 行向量（每列均值）
mean(r, 2)   % 返回 5×1 列向量（每行均值）
mean(r(:))   % 返回标量（整体均值）← 推荐
```

## 🟡 概念模糊

### 4. legend 的正确用法

```matlab
% ❌
legend(LineWidth=1);

% ✅
legend('显示文本', 'Location', 'best');
```

### 5. MATLAB 引号兼容性

```matlab
% ❌ R2018b~R2022a 可能不支持
mean(r, "all")    

% ✅ 全兼容
mean(r(:))
```
