# 扩频通信概念解析

> 这是你学 GNSS 信号时最核心的概念。

---

## 1. 为什么北斗信号那么弱还能被收到？

卫星在天上 20,000 公里，信号到地面时功率 ≈ -130dBm（**比 WiFi 弱十亿倍**）。

```
一个直观比喻：
  北斗信号 = 北京的一根针掉地上
  白噪声   = 你家楼下菜市场的噪音
  扩频增益 = 给你一个超级耳朵，能听出那根针的声音
```

## 2. 扩频 = 把信号"铺开"

原始数据（50bps）→ 乘以 PRN 码（2.046Mcps）→ 带宽从 50Hz 扩展到 4MHz

```
    ┌──────┐     × PRN码     ┌─────────────┐
    │ 数据  │ ──────────────→ │ 扩频后的信号 │     ← 带宽宽了 40,000 倍
    │ 50Hz  │                 │ 4MHz 带宽   │
    └──────┘                 └─────────────┘
                                  ↓ 发射
                             卫星天线 → 地面
```

## 3. 为什么叫"扩频增益"

接收机知道 PRN 码，把收到的信号乘上同样的 PRN 码：

```
扩频增益 = 10 × log10(码片数)

北斗 B1I:  2046 chips
增益 = 10 × log10(2046) ≈ 33 dB
```

33dB 意味着：信号即使比噪声弱 1000 倍，解扩后还是能恢复出来。

## 4. 扩频的三个好处（面试常考）

| 好处 | 说明 | 面试怎么说 |
|------|------|-----------|
| ① 抗干扰 | 窄带干扰只能破坏部分频谱，扩频增益能恢复 | "干扰容忍度 ≈ 扩频增益" |
| ② 多址接入 | 不同卫星用不同 PRN 码（CDMA） | "一颗卫星一个码，接收机能区分" |
| ③ 抗多径 | 延迟的信号和直达信号相关性低 | "自相关特性决定了多径分辨能力" |
