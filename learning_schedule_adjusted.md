## 为什么这个方案可行

### 轻量章 vs 重量章分布

```
轻量（半天可搞定）：  C 06 strings / 07 ringbuffer / 09 pool-allocator
                     MATLAB 03 signal-gen / 04 file-io

重量（需 1~2 天）：  C 08 nmea-parser（阶段1交付物）
                     MATLAB 02 FFT（新概念多）

中间的：              C 10 state-machine
                     MATLAB 05 nmea-verify
```

### 每天 2~3 小时就够了

```
中午在公司（30-45min）：
  MATLAB 跑一章 → 公司有环境，直接 F5

晚上回家（60-90min）：
  C 学半章 + 做题
  周末集中补重量章
```
