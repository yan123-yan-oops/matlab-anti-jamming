# 10 最终集成项目
# 这是整个学习工程的毕业设计

---

## 项目目标

把前面 01~09 的所有知识串起来，完成一个**北斗抗干扰全链路仿真**。

```
信号生成 → 加干扰 → 抗干扰 → 捕获验证 → 性能报告
```

---

## 要求

### 功能要求

1. 生成北斗 B1I 信号（含 PRN 码 + BPSK 调制）
2. 叠加三种干扰：CW + 宽带噪声 + 脉冲
3. 分别用三种抗干扰算法处理
4. 对比处理前后的相关峰
5. 输出性能报告（表格 + 图表）
6. 生成 KML 轨迹文件（和 C 的 NMEA 解析器联动）

### 代码要求

1. 模块化：每个功能写成独立的函数
2. 可配置：JSR、干扰频率、算法参数等顶层可调
3. 可复现：rng 固定种子
4. 文档：注释清晰，面试官能看懂

### 交付物

```
10-final-project/
├── run_full_chain.m      ← 主入口
├── output/                ← 运行结果（自动生成）
│   ├── before_vs_after.png   ← 处理前后对比图
│   ├── performance_report.txt ← 性能报告
│   └── gps_track.kml         ← Google Earth 轨迹
└── EXERCISES.md            ← 本文件
```
